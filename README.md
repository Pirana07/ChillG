# Clicker game

Idle Mining Clicker Game

An idle / clicker-style mining game built with Unity.
Click to earn resources, hire miners to automate your income, and upgrade your production system. Perform rebirths to reset progress in exchange for powerful multipliers, unlocking evolution-based upgrades that significantly boost efficiency and push progression further with every run. (More systems coming soon.)
