# ChillG

Idle Mining Clicker Game

An idle / clicker-style mining game built with Unity.
Click to earn resources, hire miners to automate income, and upgrade your production system. Use rebirths to reset progress in exchange for powerful multipliers and unlock new evolution-based upgrades that push progression further each run.(for now)
